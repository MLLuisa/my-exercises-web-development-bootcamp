Tecnologías web

Existen numerosas tecnologías web, algo que si bien es una ventaja en el diseño y desarrollo de sitio web, puede llegar a complicar la elección o la toma de decisión de cuál utilizar y para qué. Por eso es imprescindible explicar en detalle el proyecto que se quiere desarrollar, las funciones que necesitará el desarrollo web, y en base a eso, dejarse a asesorar por los desarrolladores a la hora de determinar qué tecnologías web se usarán.Como hemos mencionado, las tecnologías web avanzan rápidamente y es necesario estar al día de las tecnologías más actuales. Las diferentes tecnologías permiten crear interfaces de usuario y establecer las comunicaciones con el servidor, además de implementar comportamientos de la web en el servidor. Pero, antes de explicar las tecnologías más actuales, hay que distinguir entre tecnologías backend o que trabajan del lado del servidor y tecnologías frontend o que trabajan del lado del cliente.
